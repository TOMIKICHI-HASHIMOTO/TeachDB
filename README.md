# TeachDB

## Build Memory Layers for Humanity

TeachDB is a platform for building **Memory Layers** of historical figures.

A Memory Layer is a structured knowledge repository that preserves everything known about one person.

Rather than training a new AI for every individual, TeachDB stores a person's accumulated knowledge and allows modern Large Language Models (LLMs) to reason from that knowledge.

TeachDB does not replace GPT.

Instead, it separates knowledge from reasoning.

```
Memory Layer
        │
        ▼
      RAG
        │
        ▼
Large Language Model
        │
        ▼
    Reasoning
```

As LLMs evolve, reasoning improves.

The Memory Layer remains a long-term knowledge asset.

---

# What is a Memory Layer?

A Memory Layer is a digital representation of one person's accumulated knowledge.

Examples include:

- Osamu Dazai Memory Layer
- Gentaku Otsuki Memory Layer
- Kenji Miyazawa Memory Layer

A Memory Layer may contain:

- Books
- Letters
- Manuscripts
- Diaries
- Historical records
- Public speeches
- Other primary sources

TeachDB focuses on preserving knowledge.

Interpretation is generated by the LLM.

---

# Why TeachDB?

Human knowledge is scattered across books, archives, manuscripts, museums, and libraries.

Understanding one historical figure often requires reading thousands of pages.

TeachDB organizes this knowledge into a single Memory Layer.

Instead of searching documents, users interact with accumulated human knowledge.

# Why not just RAG?

RAG is a retrieval technology.

TeachDB defines **what is being retrieved**.

TeachDB introduces the concept of a Memory Layer:

a structured knowledge repository dedicated to one individual.

The Memory Layer is the long-term asset.

RAG and LLMs will continue to evolve.

The Memory Layer remains.

---

# Long-term Vision

TeachDB is not the final destination.

The long-term vision is **Kenjindo (House of Wisdom).**

Kenjindo is a digital library where every historical figure has their own Memory Layer.

Users will not simply search information.

They will learn directly from humanity's accumulated knowledge.
